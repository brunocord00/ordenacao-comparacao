# Algoritmos de Ordenação – SelectionSort e InsertionSort

Atividade da disciplina de Estrutura de Dados e Complexidade de Algortimo, com o objetivo de implementar e testar os algoritmos de ordenação **Selection Sort** e **Insertion Sort**.

O programa lê arquivos contendo números, realiza a ordenação e mostra o resultado no terminal.

## O que foi implementado

* Selection Sort
* Insertion Sort
* Leitura dos números a partir de arquivos de entrada
* Execução via linha de comando
* Makefile para automatizar a compilação e a execução

## Como compilar

Dentro da pasta do projeto, executar:

make

## Como executar

Para rodar o SelectionSort:

make selection

Para rodar o InsertionSort:

make insertion

## Estrutura do projeto

src → contém os códigos dos algoritmos
instancias-numericas → arquivos usados para teste
Makefile → facilita a compilação e execução

## Observação

Os algoritmos foram implementados seguindo a lógica do conteúdos estudados. Os testes foram feitos utilizando diferentes arquivos de entrada para verificar o funcionamento da ordenação.
